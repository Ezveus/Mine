MESSAGE ::= REQUEST = { ARGUMENTS[, ARGUMENTS]* }

REQUEST = AUTHENTICATE | SIGNUP | EXEC | INSERT | MOVE | BACKSPACE | DELETE | SYNC

ARGUMENTS ::= " VALUE " : " VALUE "

VALUE ::= .

Command List and Arguments Sent by Clients :
-> AUTHENTICATE : ReqId name pass

-> SIGNUP : ReqId name pass email (website)

-> EXEC : ReqId buffer command (args[array[value]])

-> INSERT : ReqId buffer text

-> MOVE : ReqId buffer direction[up | right | down | left | origin | end] number[integer]

-> BACKSPACE : ReqId buffer number[integer]

-> DELETE : ReqId buffer number[integer]

-> LOAD : ReqId path size[integer] line[integer] port[integer]


Command List and Arguments sent by Server :

-> SYNC : array[(buffer diff)]

-> RESPONSE : id status[integer] {object}

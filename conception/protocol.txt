MESSAGE ::= REQUEST = { ARGUMENTS[, ARGUMENTS]* }

REQUEST = AUTHENTICATE | SIGNUP | EXEC | INSERT | MOVE | BACKSPACE | DELETE | SYNC

ARGUMENTS ::= " VALUE " : " VALUE "

VALUE ::= .

Command List and Arguments :
-> AUTHENTICATE : name pass

-> SIGNUP : name pass email (website)

-> EXEC : buffer command (args[array[value]])

-> INSERT : buffer text

-> MOVE : buffer direction[up | right | down | left | origin | end] number[integer]

-> BACKSPACE : buffer number[integer]

-> DELETE : buffer number[integer]

-> SYNC : array[(buffer diff)]

-> FILE : path size content
